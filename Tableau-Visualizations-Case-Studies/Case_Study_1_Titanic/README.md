# Case Study 1 – Titanic Dataset

### Objective
To analyze the Titanic passenger dataset and visualize survival trends, demographic patterns, and class-wise differences.

### Files Included
- `titanic.csv` – Dataset
- `Case Study 1 Titanic.twb` – Tableau workbook
- `Case Study 1 Titanic.pdf` – Visualization outputs
- `CASE_STUDY_1_TITATNIC.docx` – Instructions

### Key Visualizations
1. Passenger count by gender
2. Survival count by gender
3. Survival rate by passenger class
4. Age distribution histogram
5. Average fare by class
6. Scatter plot (Fare vs Age)
7. Passenger count by embarkation port
8. Survival heatmap by gender and class
9. Box plot (Age vs Survival)
10. Pie chart (Gender distribution)

### Insights
- Survival rates were **highest for 1st class passengers (62%)** and **female passengers (≈ 92%)**.
- **Class 3** passengers had the lowest survival rate (24%).
- Fare increased significantly with class.

### Tools Used
- Tableau Desktop
- CSV Import
- Aggregation (SUM, AVG, COUNT)
